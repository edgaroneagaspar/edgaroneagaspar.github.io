Research areas

The site is designed as a compact intellectual profile rather than a web version of a CV. It presents the main research threads, publications, talks, teaching and academic service in one coherent structure.

Semantics & pragmatics

Questions, exhaustivity, alternatives

Work on how linguistic meaning encodes and constrains inquiry, alternatives, focus and contextual resolution.

Discourse

Grammar and discourse structure

Research on discourse referents, appositives, topics, conditionals, rhetorical relations and anaphora.

Interdisciplinary work

Narrative, experiments, variation

Work connecting formal linguistics to narrative theory, experimental methods, sign language, and diachronic variation.

Website architecture

This prototype already contains the multilingual shell. English pages contain real content; German, Romanian and Hungarian pages are placeholders for later expansion.

Single data source

Publications are read from one JSON file and then displayed by year, type or topic. This avoids maintaining separate lists manually.

Static and durable

No build system is required. The files can be copied directly to GitHub Pages and edited with ordinary HTML, CSS and JSON.
